== Description ==

**QUICK VIEW EXTENSION FOR WOOCOMMERCE ALLOWS CUSTOMERS TO VIEW PRODUCTS WITHOUT LEAVING THE SHOP PAGE.**

MWB Quick View for WooCommerce plugin helps People can easily choose if they can actually see and adding this functionality on your WooCommerce store will allow your customers to view the picture gallery of products. The users can quickly glance at products and their descriptions without having to wait for the single product page to upload.

The woo quick view extension is a tool that helps you in converting prospects into customers by enhancing their user experience. The product quick view extension comes with enhancements to customize the modal window as per your business niche.

Using our quick view extension, you can easily add user-friendly features for your customers. You get to choose the details you want to display in the product description in the modal window.

You can also improve the aesthetics of your window by choosing an animation from the given three options. The extension allows you to customize the quick view button and closing button for text and color respectively.

== WITH QUICK VIEW FOR WOOCOMMERCE PLUGIN, YOU CAN: ==

* Add animation for the modal window.
* Edit production description to be displayed on the quick view window
* Disable the quick view button for mobile view.
* Change the text of the quick view button
* Change the text color and background color of the quick view button
* Choose the products or categories to exclude from the quick view option.


== WHAT IS INCLUDED IN THE FREE QUICK VIEW FOR WOOCOMMERCE PLUGIN: ==

**1) Text In The Quick View Button:** 
You can write suitable text for the quick view button on the shop page for the front end. 

**2) Animation For The Modal Window:**
The quick view plugin provides for 3 default animations to choose from. These animation effects are for the quick view pop-up. You can preview the effect in your dashboard and save your choice accordingly.

**3) Customization Of Modal Window:**
Predefine the product details you want to mention in the product description such as product and gallery images, SKU, title, price, and so on.

**4) Add To Cart Button**
You can also add the Add to Cart button in the quick view pop-up window. When prompted by the user in the frontend, the default messages will be displayed, as on the cart page and shop page. 

**5) View Details Button**
The pop-up gives the option of a ‘View Details’ button. When the users will click on this option, they will be redirected to the single product page. Here, they can view the complete description of the product available in your store.

**6) Customise Closing Button:**
The extension allows you to customize the closing button for its background color. You can also set the background color of the closing button for when the pointer hovers over the button. 

**7) Customise Quick View Button:**
You can set both- the text and the background color of the quick view button for your shop page using the quick view WooCommerce extension.

== Installation ==
 
= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of **Quick View for WooCommerce**, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type **"MWB Quick View for WooCommerce"** and click Search Plugins. Once you've found our Quick View for WooCommerce plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking **"Install Now"**.

= Manual installation =

The manual installation method involves downloading our Quick View for WooCommerce and uploading it to your webserver via your favorite FTP application. The WordPress codex contains [**Instructions on how to do this here**](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
